CODIGO DE HTML

¿Cómo usar los códigos html?

Para crear una página web completa debes hacer uso de documentos html para poder tener organizados todos los códigos que vayas a usar. Este documento es un archivo con una extensión .html y puede realizarse en el editor de textos de tu preferencia.

¿Tipos de etiquetas de html?

A grandes rasgos, en un documento HTML existen dos tipos de etiquetas HTML: las etiquetas que contienen fragmentos de texto (se utilizan para dar significado a esas palabras o fragmentos) y las etiquetas que agrupan conjunto de información (fragmentos de texto y/u otras etiquetas).
